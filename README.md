## Installation

### RubyGems

```
$ gem install fluent-plugin-windows-evtx
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-windows-evtx"
```

And then execute:

```
$ bundle
```

You can generate configuration template:

```
$ fluent-plugin-config-format input windows-evtx
```
